Fe3O4上RWGSの最小素反応ネットワーク

この章では、Fe3O4 上の RWGS (reverse water-gas shift) を記述するための最小素反応ネットワークを定義する。目的は、CO2 と H2 から CO と H2O が生成する過程を、site balance と elemental mass balance を保った形で microkinetic model に落とし込むことである。

本章で扱う Fe3O4 は、MVP では RWGS-active phase として扱う。FeCx site、C-C coupling、chain growth はこの章では実装しない。FeCx-FT は将来拡張として必要になる可能性があるが、ここでは Fe3O4-RWGS の最小モデルに集中する。

1 学習目標

この章を終えると、次のことができるようになる。

  • Fe3O4-RWGS MVP で使う気相種と表面種を列挙する。
  • R1 から R7 までの最小素反応機構を説明する。
  • 各素反応の Type と Role in the model を区別する。
  • 表面種 *, CO2*, H*, CO*, O*, OH*, H2O* の役割を説明する。
  • この最小モデルで説明できる範囲と、説明できない範囲を区別する。

2 この章で扱うモデル範囲

この章で扱う総括反応は RWGS である。

CO2 + H2 <=> CO + H2O

LaTeX では次のように書く。

\[ \mathrm{CO_2 + H_2 \rightleftharpoons CO + H_2O} \]

モデルは単一種類の表面サイト * を仮定する。したがって、Fe site と O site を分ける multi-site model、oxygen vacancy を明示するモデル、FeCx 上の FT model は扱わない。この単純化により、最初に site balance、mass balance、steady-state coverage の計算に集中できる。

3 気相種リスト

MVP で扱う気相種は4つだけである。

Species 役割
CO2 反応物。RWGS の炭素源および酸素源。
H2 反応物。表面水素 H* の供給源。
CO 生成物。Fe3O4-RWGS MVP で追跡する主な炭素含有生成物。
H2O 生成物。表面酸素の除去先。

4 表面種リスト

MVP で扱う表面種は次の7つである。

Surface species 役割
* 空の表面サイト。すべての吸着・脱離・表面反応の site balance に入る。
CO2* 吸着 CO2。CO2 活性化の入口。
H* 吸着水素。O* と OH* の水素化に使われる。
CO* 吸着 CO。CO desorption を通じて気相 CO になる。
O* 表面酸素。CO2* の解離で生じ、OH* へ水素化される。
OH* 表面ヒドロキシ種。H* と反応して H2O* になる。
H2O* 吸着水。脱離して気相 H2O になる。

この表面種セットは、上の7種類だけに限定する。別経路に由来する中間体は重要になり得るが、本MVPでは余分な素反応を追加しない。

4.1 COOH* と HCOO* を含めない理由

RWGS では、CO2* の直接解離経路だけでなく、COOH* を経由する carboxyl 経路や、HCOO* を経由する formate 経路も候補になり得る。

本 MVP では、Fe3O4-RWGS の direct dissociation / redox 型の最小モデルを作ることを目的とするため、COOH* と HCOO* は明示的な表面種として含めない。

したがって、本章の反応機構は COOH* や HCOO* が不要であると断定するものではなく、最小モデルとして意図的に省略しているものである。これらの中間体は、将来の拡張モデルで direct / COOH / HCOO 経路を比較するときに追加する。

5 7ステップ最小反応機構

Fe3O4-RWGS MVP では、反応 ID を必ず R1 から R7 とする。表記は以後も変えない。

Reaction ID Elementary step Type Role in the model
R1 CO2 + * <=> CO2* Adsorption / desorption CO2 を表面へ取り込む入口。
R2 H2 + 2* <=> 2H* Dissociative adsorption / recombination H2 から表面水素 H* を作る。
R3 CO2* + * <=> CO* + O* Surface dissociation / recombination 吸着 CO2 を CO* と O* に分ける。
R4 CO* <=> CO + * Desorption / adsorption 気相 CO を生成する出口。
R5 O* + H* <=> OH* + * Surface hydrogenation / dehydrogenation 表面酸素 O* を OH* に変換する。
R6 OH* + H* <=> H2O* + * Surface hydrogenation / dehydrogenation OH* を H2O* に変換する。
R7 H2O* <=> H2O + * Desorption / adsorption 気相 H2O を生成し、空サイトを再生する。

同じ7ステップを LaTeX で書くと次のようになる。

\[ \mathrm{R1:\ CO_2 + * \rightleftharpoons CO_2*} \]

\[ \mathrm{R2:\ H_2 + 2* \rightleftharpoons 2H*} \]

\[ \mathrm{R3:\ CO_2* + * \rightleftharpoons CO* + O*} \]

\[ \mathrm{R4:\ CO* \rightleftharpoons CO + *} \]

\[ \mathrm{R5:\ O* + H* \rightleftharpoons OH* + *} \]

\[ \mathrm{R6:\ OH* + H* \rightleftharpoons H_2O* + *} \]

\[ \mathrm{R7:\ H_2O* \rightleftharpoons H_2O + *} \]

6 各表面種の役割

* は空サイトであり、吸着、脱離、表面反応のすべてに関わる。被覆率の合計は次の site balance を満たす。

\[ \theta_* + \theta_{\mathrm{CO_2}} + \theta_{\mathrm{H}} + \theta_{\mathrm{CO}} + \theta_{\mathrm{O}} + \theta_{\mathrm{OH}} + \theta_{\mathrm{H_2O}} = 1 \]

ここで \(\theta_{\mathrm{CO_2}}\) は CO2* の被覆率を表す。同様に、\(\theta_{\mathrm{H}}\) は H*、\(\theta_{\mathrm{CO}}\) は CO*、\(\theta_{\mathrm{O}}\) は O*、\(\theta_{\mathrm{OH}}\) は OH*、\(\theta_{\mathrm{H_2O}}\) は H2O* の被覆率を表す。

CO2* は CO2 activation の入口である。R1 で生成し、R3 で CO* と O* に変換される。

H* は H2 dissociation により生成する表面水素である。R5 と R6 で表面酸素種を水素化する。

CO* は CO 生成直前の表面中間体である。R4 で脱離して CO になる。

O* は CO2* の解離で生じる表面酸素である。O* が蓄積すると空サイトが減るため、R5 と R6 を通じた水素化と水生成が重要になる。

OH* は O* の水素化で生じる表面ヒドロキシ種である。さらに H* と反応して H2O* になる。

H2O* は水生成直前の吸着水である。R7 で脱離して気相 H2O になり、空サイト * を再生する。

7 この最小モデルで説明できること

この最小モデルで説明できるのは、Fe3O4 を RWGS-active phase と見なしたときの基本的な表面反応である。

  • CO2 adsorption と CO2 activation
  • H2 dissociation による表面水素供給
  • CO* の脱離による CO 生成
  • O* から OH*、H2O* を経た水生成
  • site balance を満たす steady-state surface coverage
  • RWGS 範囲での CO 生成速度と H2O 生成速度

この範囲では、CO2 から CO と H2O が生成する過程を、最小限の表面種で追跡できる。

8 この最小モデルで説明できないこと

このモデルは、次の現象を説明するためのものではない。

  • CH4 生成の詳細機構
  • C2+ hydrocarbon formation
  • C-C coupling
  • chain growth と chain termination
  • ASF distribution
  • Fe3O4 から FeCx への dynamic phase transformation
  • FeCx site 上の FT reaction network
  • COOH* や HCOO* を含む competing pathway の比較

これらを扱うには、別の species set、別の elementary step、別の検証項目が必要である。FeCx については将来拡張として扱うが、本章では実装しない。

9 C2+生成に関する注意

C2+生成は、この MVP の範囲外である。Fe3O4-RWGS model で CO 生成を説明できても、それは Fe3O4 単独で C2+ 炭化水素生成を説明できることを意味しない。

C2+選択率を扱う段階では、FeCx-FT、C-C coupling、chain growth、termination を含む別モデルを定義する必要がある。そのときも、Fe3O4-RWGS と FeCx-FT の役割を混同しない。

10 この章の要点

  • Fe3O4-RWGS MVP では、気相種を CO2, H2, CO, H2O に限定する。
  • 表面種は *, CO2*, H*, CO*, O*, OH*, H2O* の7種類である。
  • 素反応は R1 から R7 の7ステップだけである。
  • このモデルは RWGS による CO と H2O の生成を扱う。
  • C2+生成、ASF分布、FeCx-FT は将来拡張であり、本MVPでは実装しない。

11 確認問題

  1. Fe3O4-RWGS MVP で扱う4つの気相種を挙げ、それぞれの役割を説明せよ。
  2. R3: CO2* + * <=> CO* + O* が site balance を満たす理由を説明せよ。
  3. O*, OH*, H2O* はそれぞれ RWGS model の中でどのような役割を持つか。
  4. この最小モデルで C2+生成を説明できない理由を述べよ。
  5. R7 が空サイト * の再生にどのように関わるか説明せよ。

12 計算演習

  1. R1 から R7 について、反応前後の表面サイト数を数え、すべての反応で site balance が保存されることを確認せよ。
  2. R3, R5, R6 について、C、H、O の elemental mass balance を確認せよ。
  3. 任意の被覆率セットを仮定し、次の site balance が1になるように正規化せよ。

\[ \theta_* + \theta_{\mathrm{CO_2}} + \theta_{\mathrm{H}} + \theta_{\mathrm{CO}} + \theta_{\mathrm{O}} + \theta_{\mathrm{OH}} + \theta_{\mathrm{H_2O}} = 1 \]
