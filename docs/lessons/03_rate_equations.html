速度式・サイトバランス・熱力学整合性

この章では、第2章で定義した Fe3O4-RWGS 最小7ステップ機構に対して、素反応速度式、サイトバランス、定常状態方程式、熱力学整合性を整理する。目的は、数値計算を始める前に、どの量を独立に与え、どの量を保存則や平衡関係から制約すべきかを明確にすることである。

この章では PFR、パラメータ推定、FeCx-FT の詳細は扱わない。対象は Fe3O4 上 RWGS の最小モデルに限定する。

1 学習目標

この章を終えると、次のことができるようになる。

  • 素反応ごとの前進速度、逆反応速度、正味速度を定義する。
  • mass-action kinetics に基づいて R1 から R7 の速度式を書く。
  • Fe3O4-RWGS 最小モデルの site balance を書く。
  • 表面被覆率の steady-state equations を立てる。
  • CO2、H2、CO、H2O の正味生成速度を R1-R7 の速度で表す。
  • 詳細釣り合いと熱力学整合性の意味を説明する。
  • 逆反応速度定数を自由にフィットすべきでない理由を説明する。

2 前提となるR1-R7機構

本章では、第2章と同じ反応 ID と素反応だけを使う。

Reaction ID Elementary step
R1 CO2 + * <=> CO2*
R2 H2 + 2* <=> 2H*
R3 CO2* + * <=> CO* + O*
R4 CO* <=> CO + *
R5 O* + H* <=> OH* + *
R6 OH* + H* <=> H2O* + *
R7 H2O* <=> H2O + *

余分な素反応は追加しない。

3 素反応速度式の一般形

素反応 Rj の正味速度 \(r_j\) は、前進速度 \(r_{j,f}\) と逆反応速度 \(r_{j,r}\) の差として定義する。

\[ r_j = r_{j,f} - r_{j,r} \]

ここで \(r_j > 0\) は、表に書いた左辺から右辺へ進む向きを意味する。

4 前進速度と逆反応速度

mass-action kinetics では、前進速度と逆反応速度を reactant activity の積として書く。

\[ r_{j,f} = k_{j,f} \prod_i a_i^{\nu_{ij,f}} \]

\[ r_{j,r} = k_{j,r} \prod_i a_i^{\nu_{ij,r}} \]

ここで \(k_{j,f}\) と \(k_{j,r}\) は前進・逆反応速度定数、\(a_i\) は species \(i\) の activity、\(\nu_{ij,f}\) と \(\nu_{ij,r}\) は前進側・逆反応側での反応次数である。

この MVP では、気相種の activity を分圧 \(P_i\)、表面種の activity を被覆率 \(\theta_i\) で近似する。

5 mass-action kinetics

たとえば R1 は次の吸着・脱離反応である。

R1: CO2 + * <=> CO2*

LaTeX では次のように書く。

\[ \mathrm{R1:\ CO_2 + * \rightleftharpoons CO_2*} \]

このとき、正味速度は次のようになる。

\[ r_1 = k_{1,f} P_{\mathrm{CO_2}} \theta_* - k_{1,r} \theta_{\mathrm{CO_2}} \]

表面種の添字では、簡潔さのため CO2* を \(\theta_{\mathrm{CO_2}}\) と書く。以後も、\(\theta_{\mathrm{H}}\) は H*、\(\theta_{\mathrm{O}}\) は O*、\(\theta_{\mathrm{H_2O}}\) は H2O* の被覆率を表す。

6 Fe3O4-RWGS最小モデルのサイトバランス

本MVPでは、単一種類の表面サイト * を仮定する。したがって、全表面被覆率の和は1である。

\[ \theta_* + \theta_{\mathrm{CO_2}} + \theta_{\mathrm{H}} + \theta_{\mathrm{CO}} + \theta_{\mathrm{O}} + \theta_{\mathrm{OH}} + \theta_{\mathrm{H_2O}} = 1 \]

この式は、空サイト * も表面種として数えることを意味する。空サイトを site balance に含めないと、吸着・脱離反応で表面サイト数が保存されない。

7 R1からR7の速度式例

第2章の7ステップに対して、mass-action form の正味速度を次のように書く。

\[ r_1 = k_{1,f} P_{\mathrm{CO_2}} \theta_* - k_{1,r} \theta_{\mathrm{CO_2}} \]

\[ r_2 = k_{2,f} P_{\mathrm{H_2}} \theta_*^2 - k_{2,r} \theta_{\mathrm{H}}^2 \]

\[ r_3 = k_{3,f} \theta_{\mathrm{CO_2}} \theta_* - k_{3,r} \theta_{\mathrm{CO}} \theta_{\mathrm{O}} \]

\[ r_4 = k_{4,f} \theta_{\mathrm{CO}} - k_{4,r} P_{\mathrm{CO}} \theta_* \]

\[ r_5 = k_{5,f} \theta_{\mathrm{O}} \theta_{\mathrm{H}} - k_{5,r} \theta_{\mathrm{OH}} \theta_* \]

\[ r_6 = k_{6,f} \theta_{\mathrm{OH}} \theta_{\mathrm{H}} - k_{6,r} \theta_{\mathrm{H_2O}} \theta_* \]

\[ r_7 = k_{7,f} \theta_{\mathrm{H_2O}} - k_{7,r} P_{\mathrm{H_2O}} \theta_* \]

これらは教材用の最小表現であり、activity coefficient や coverage dependence を含む拡張はここでは扱わない。

8 各表面被覆率の定常状態方程式

表面種 \(i\) の被覆率が定常状態にあるとき、生成速度と消費速度の和は0になる。

\[ \frac{d\theta_i}{dt} = \sum_j \nu_{ij} r_j = 0 \]

ここで \(\nu_{ij}\) は、反応 Rj における表面種 \(i\) の化学量論係数である。生成側を正、消費側を負とする。

Fe3O4-RWGS MVP では、空サイト * を site balance で決め、残りの表面種について定常状態方程式を立てる。

\[ \frac{d\theta_{\mathrm{CO_2}}}{dt} = r_1 - r_3 = 0 \]

\[ \frac{d\theta_{\mathrm{H}}}{dt} = 2r_2 - r_5 - r_6 = 0 \]

\[ \frac{d\theta_{\mathrm{CO}}}{dt} = r_3 - r_4 = 0 \]

\[ \frac{d\theta_{\mathrm{O}}}{dt} = r_3 - r_5 = 0 \]

\[ \frac{d\theta_{\mathrm{OH}}}{dt} = r_5 - r_6 = 0 \]

\[ \frac{d\theta_{\mathrm{H_2O}}}{dt} = r_6 - r_7 = 0 \]

これらの式と site balance を同時に満たす被覆率が、steady-state surface coverage である。

9 CO2, H2, CO, H2Oの正味生成速度

気相種の正味生成速度は、対応する吸着・脱離ステップから求められる。ここでは、気相に生成される向きを正とする。

\[ R_{\mathrm{CO_2}} = -r_1 \]

\[ R_{\mathrm{H_2}} = -r_2 \]

\[ R_{\mathrm{CO}} = r_4 \]

\[ R_{\mathrm{H_2O}} = r_7 \]

RWGS の正味進行が支配的であれば、\(R_{\mathrm{CO}}\) と \(R_{\mathrm{H_2O}}\) は生成速度、\(R_{\mathrm{CO_2}}\) と \(R_{\mathrm{H_2}}\) は消費速度として解釈される。

10 詳細釣り合い

平衡状態では、各素反応について前進速度と逆反応速度が等しくなる。

\[ r_{j,f} = r_{j,r} \]

したがって、平衡定数 \(K_j\) は速度定数の比と対応する。

\[ K_j = \frac{k_{j,f}}{k_{j,r}} \]

この関係は、速度定数が熱力学と無関係に選べるわけではないことを示している。反応全体の平衡定数や各素反応の自由エネルギー差が分かっている場合、\(k_{j,f}\) と \(k_{j,r}\) の比はそれらと整合していなければならない。

11 逆反応速度定数を独立にフィットすべきでない理由

前進速度定数と逆反応速度定数をすべて自由なパラメータとして独立にフィットすると、見かけ上はデータに合っても、熱力学的に矛盾したモデルになることがある。

たとえば、ある素反応について \(K_j\) が既知または推定可能であれば、次の関係を満たす必要がある。

\[ k_{j,r} = \frac{k_{j,f}}{K_j} \]

つまり、\(k_{j,f}\) を決めたあとに \(k_{j,r}\) を完全に自由に動かすと、詳細釣り合いを壊す可能性がある。microkinetic model では、速度論的フィットと熱力学整合性を同時に満たすようにパラメータを扱う必要がある。

この教材では、逆反応速度定数を自由にフィットしてよいとは扱わない。必要な熱力学データが不足している場合は、仮定を明示し、[citation needed] を残す。

12 この章の要点

  • 正味速度は \(r_j = r_{j,f} - r_{j,r}\) で定義する。
  • mass-action kinetics では、速度は分圧と表面被覆率の積で表される。
  • Fe3O4-RWGS MVP の site balance は、*, CO2*, H*, CO*, O*, OH*, H2O* の和が1である。
  • 定常状態では、各表面種について \(\frac{d\theta_i}{dt}=0\) を満たす。
  • 詳細釣り合いにより、\(K_j = k_{j,f}/k_{j,r}\) が成り立つ。
  • 逆反応速度定数を熱力学と無関係に自由フィットするのは避ける。

13 確認問題

  1. 正味速度 \(r_j\) と前進速度 \(r_{j,f}\)、逆反応速度 \(r_{j,r}\) の関係を書け。
  2. R2: H2 + 2* <=> 2H* の速度式を書き、\(\theta_*^2\) が現れる理由を説明せよ。
  3. Fe3O4-RWGS MVP の site balance を書け。
  4. CO* の定常状態方程式が \(r_3 - r_4 = 0\) になる理由を説明せよ。
  5. 逆反応速度定数をすべて独立にフィットすると、どのような問題が起こり得るか説明せよ。

14 計算演習

  1. 任意の被覆率セットについて site balance を計算し、1になるように正規化せよ。
  2. 与えられた \(k_{1,f}\)、\(k_{1,r}\)、\(P_{\mathrm{CO_2}}\)、\(\theta_*\)、\(\theta_{\mathrm{CO_2}}\) から \(r_1\) を計算せよ。
  3. 仮に \(K_4\) と \(k_{4,f}\) が与えられているとき、\(k_{4,r}=k_{4,f}/K_4\) を使って逆反応速度定数を求めよ。
