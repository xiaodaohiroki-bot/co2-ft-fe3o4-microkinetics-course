DFTから速度定数を作る手順

この章では、Fe3O4-RWGS 最小モデルを microkinetic model として使うために、DFT 計算から速度定数を作るまでの流れを整理する。対象は第2章で定義した R1-R7 のみであり、FeCx-FT、C2+ 生成、ASF 分布、動的相変化は扱わない。

ここで示すのは、実験値や文献値ではなく、計算計画を立てるための手順である。実際の数値を使う段階では、計算条件、収束性、表面構造、熱補正、標準状態をすべて明示する必要がある。

1 学習目標

この章を終えると、次のことを説明できるようになる。

  • microkinetic model に必要な DFT 計算の種類を列挙する。
  • Fe3O4-RWGS R1-R7 の各ステップに必要な吸着エネルギー、反応エネルギー、活性化障壁を対応づける。
  • DFT エネルギーから自由エネルギー、平衡定数、速度定数へ変換する流れを説明する。
  • 逆反応速度定数を熱力学整合性と無関係に自由フィットしてはいけない理由を説明する。
  • MVP で扱わない COOH、HCOO、FeCx-FT を計算計画から分離する。

2 マイクロキネティクスにDFT計算が必要な理由

microkinetic model では、各素反応の速度を次のように書く。

\[ r_j = r_{j,f} - r_{j,r} \]

mass-action kinetics を使う場合、速度式そのものは第3章で定義できる。しかし、速度定数 \(k_{j,f}\) と \(k_{j,r}\) を物理的に意味のある値として与えるには、各素反応の energetics が必要である。

DFT 計算で主に求める量は、次の3つである。

Quantity 役割 microkinetic model での使い道
吸着エネルギー 気相種が表面へどれだけ安定に吸着するか 吸着・脱離平衡、表面被覆率の傾向
反応エネルギー 反応前後の安定性差 平衡定数、熱力学整合性
活性化障壁 遷移状態までの自由エネルギー差 前進・逆反応速度定数

つまり、DFT 計算は「反応式を書くため」ではなく、「反応式に入る速度定数を制約するため」に必要である。

3 Fe3O4表面モデルの準備

最初に、どの Fe3O4 表面を model surface として使うかを決める。表面を決めずに吸着エネルギーや障壁だけを集めても、同じ速度定数セットとして比較できない。

最低限、次を決めて記録する。

  • exchange-correlation functional
  • Hubbard \(U\) の扱い
  • spin polarization と初期磁気モーメント
  • bulk Fe3O4 の格子定数と磁気状態
  • surface orientation
  • surface termination
  • slab thickness
  • vacuum thickness
  • 固定層と緩和層
  • k-point mesh
  • energy cutoff
  • dipole correction の有無

Fe3O4 は酸化数、磁気状態、表面終端に敏感である。したがって、bulk の妥当性を確認してから slab を作る。Fe3O4(001) のように複数の終端や再構成が候補になる面では、どの termination を採用したかを計算条件として明示する必要がある [citation needed]。

4 気相分子計算

Fe3O4-RWGS MVP では、気相種は次の4つである。

Gas species 用途
CO2 CO2 吸着、CO2 消費速度、炭素源
H2 H2 解離吸着、表面 H* 供給
CO CO 脱離、CO 生成速度
H2O H2O 脱離、酸素除去先

気相分子では、電子エネルギーだけでなく、温度依存の熱補正も必要になる。特に、気相分子では並進、回転、振動の寄与が大きいため、表面吸着種と同じ扱いにしない。

気相分子 \(i\) の自由エネルギーは、概念的には次の形で整理する。

\[ G_i(T, P_i) = E_{i,\mathrm{DFT}} + ZPE_i + H_{i,\mathrm{thermal}}(T) - T S_i(T, P_i) \]

ここで \(P_i\) は気相種の分圧である。標準状態をどう置くかは、吸着・脱離速度定数の単位に直接影響する。

5 吸着種計算

MVP で扱う表面種は次の7つである。

Surface species DFTで確認すること
* clean surface の基準エネルギー
CO2* CO2 の吸着構造、曲がり、電荷移動の有無
H* H2 解離後の H 吸着サイト
CO* CO 脱離前の吸着構造
O* CO2 解離後の表面酸素サイト
OH* O* 水素化後のヒドロキシ構造
H2O* H2O 脱離前の吸着水構造

吸着エネルギーは、たとえば CO2* について次のように定義できる。

\[ \Delta E_{\mathrm{ads}}(\mathrm{CO_2}) = E(\mathrm{CO_2*}) - E(*) - E(\mathrm{CO_2}) \]

ただし、吸着構造が複数ある場合は、最安定構造だけでなく、反応経路につながる準安定構造も候補として残す。最安定構造が必ずしも反応経路上の入口構造とは限らないためである。

COOH* と HCOO* は RWGS の競合経路候補になり得る。しかし本章では、R1-R7 の direct dissociation / redox 型 MVP に限定するため、明示的な表面種としては扱わない。これらは将来、direct / COOH / HCOO 経路を比較する拡張章で追加する。

6 R1-R7ごとに必要なDFT計算表

次の表は、R1-R7 を DFT 計算へ落とし込むときに必要な情報を整理したものである。

Reaction ID Elementary step 主に必要なDFT計算 得られる量
R1 CO2 + * <=> CO2* clean surface、gas CO2、CO2* CO2 吸着エネルギー、吸着自由エネルギー
R2 H2 + 2* <=> 2H* clean surface、gas H2、2H*、必要なら H2 dissociation TS H2 解離吸着エネルギー、解離障壁
R3 CO2* + * <=> CO* + O* CO2、CO + O*、CO2 dissociation TS CO2 解離反応エネルギー、解離障壁
R4 CO* <=> CO + * CO*、gas CO、clean surface CO 脱離エネルギー、脱離自由エネルギー
R5 O* + H* <=> OH* + * O* + H、OH、hydrogenation TS O* 水素化反応エネルギー、障壁
R6 OH* + H* <=> H2O* + * OH* + H、H2O、hydrogenation TS OH* 水素化反応エネルギー、障壁
R7 H2O* <=> H2O + * H2O*、gas H2O、clean surface H2O 脱離エネルギー、脱離自由エネルギー

この表は、必要な計算を最小限に整理するためのものである。実際には、吸着サイト、被覆率、表面終端が変わると、同じ R1-R7 でも異なるエネルギーセットになる。

7 NEB/TS計算で求める活性化障壁

反応エネルギーだけでは速度定数は決まらない。速度定数を作るには、反応物状態から遷移状態までの活性化自由エネルギーが必要である。

優先的に TS または NEB 計算を行うべきステップは、結合の生成・切断を伴う表面反応である。

Step TS/NEB の重要度 理由
R2 高 H-H 結合の切断を伴う
R3 高 CO2* の C-O 結合切断を伴う
R5 高 O-H 結合形成を伴う
R6 高 O-H 結合形成と H2O* 形成を伴う
R1, R4, R7 条件依存 吸着・脱離を障壁なし近似するか、明示的に扱うかを決める

NEB では初期状態と終状態を同じ surface cell、同じ stoichiometry、同じ固定層条件でそろえる。得られた候補遷移状態は、振動解析により虚振動が反応座標に対応しているかを確認する。

8 振動解析と自由エネルギー補正

DFT の電子エネルギー \(E_{\mathrm{DFT}}\) をそのまま速度定数に入れるのは不十分である。温度依存の速度定数を作るには、各状態の自由エネルギーをそろえる。

表面状態 \(X*\) については、概念的に次の形で整理する。

\[ G(X*, T) = E_{\mathrm{DFT}}(X*) + ZPE(X*) + H_{\mathrm{vib}}(X*, T) - T S_{\mathrm{vib}}(X*, T) \]

遷移状態では、反応座標に対応する虚振動を熱補正から除く。気相分子では、並進・回転・振動の寄与を含め、分圧依存も標準状態と一貫して扱う。

この段階で、各素反応の反応自由エネルギーを次のように作る。

\[ \Delta G_j(T) = G_{\mathrm{products},j}(T) - G_{\mathrm{reactants},j}(T) \]

また、前進方向の活性化自由エネルギーは次のように定義する。

\[ \Delta G_{j,f}^{\ddagger}(T) = G_j^{\ddagger}(T) - G_{\mathrm{reactants},j}(T) \]

逆方向についても同様に定義できる。

9 遷移状態理論による速度定数化

表面反応の速度定数は、遷移状態理論を使って概念的に次のように表せる。

\[ k_{j,f}(T) = \frac{k_{\mathrm{B}}T}{h} \exp\left( -\frac{\Delta G_{j,f}^{\ddagger}(T)}{RT} \right) \]

ここで \(k_{\mathrm{B}}\) は Boltzmann constant、\(h\) は Planck constant、\(R\) は gas constant である。

吸着ステップでは、速度定数の単位が分圧の定義に依存する。たとえば R1 の速度式を

\[ r_1 = k_{1,f} P_{\mathrm{CO_2}} \theta_* - k_{1,r}\theta_{\mathrm{CO_2}} \]

と書くなら、\(k_{1,f}\) は \(P_{\mathrm{CO_2}}\) と掛けたときに表面反応速度の単位になるように定義する必要がある。sticking coefficient を使う場合も、圧力、面積、サイト密度、分子質量の扱いを明示する。

10 熱力学整合性の確認

前進・逆反応速度定数は、反応自由エネルギーと整合していなければならない。各素反応について、平衡定数は次の関係を満たす。

\[ K_j(T) = \frac{k_{j,f}(T)}{k_{j,r}(T)} \]

一方、DFT から得た反応自由エネルギーを使えば、

\[ K_j(T) = \exp\left( -\frac{\Delta G_j(T)}{RT} \right) \]

と書ける。したがって、片方向の速度定数と \(\Delta G_j(T)\) が決まれば、もう一方の速度定数は次のように制約される。

\[ k_{j,r}(T) = \frac{k_{j,f}(T)}{K_j(T)} \]

この関係を無視して \(k_{j,f}\) と \(k_{j,r}\) を独立に調整すると、見かけ上は実験値に合っても、平衡組成や反応方向が熱力学と矛盾する可能性がある。

11 この章の要点

  • microkinetic model には、反応式だけでなく速度定数を制約する energetics が必要である。
  • Fe3O4-RWGS MVP では、R1-R7 に対応する気相分子、吸着種、遷移状態を DFT で整理する。
  • 反応エネルギーは平衡定数を、活性化自由エネルギーは速度定数を制約する。
  • 気相分子、吸着種、遷移状態では熱補正の扱いが異なる。
  • 逆反応速度定数は、熱力学整合性を満たすように前進速度定数と反応自由エネルギーから決める。
  • COOH、HCOO、FeCx-FT は重要な将来拡張候補だが、本 MVP の R1-R7 には追加しない。

12 確認問題

  1. microkinetic model で反応エネルギーだけでなく活性化障壁が必要になる理由を説明せよ。
  2. R3: CO2* + * <=> CO* + O* に必要な DFT 計算を列挙せよ。
  3. 気相分子と表面吸着種で熱補正の扱いが異なる理由を説明せよ。
  4. \(K_j = k_{j,f}/k_{j,r}\) と \(\Delta G_j\) の関係を説明せよ。
  5. COOH* と HCOO* を本 MVP に含めない理由を説明せよ。

13 計算計画演習

  1. Fe3O4(001) のある termination を仮定し、CO2*, CO*, O*, OH*, H2O* の吸着構造探索計画を作成せよ。
  2. R1-R7 のうち、NEB/TS 計算を優先すべきステップを3つ選び、その理由を説明せよ。
  3. ある素反応について \(\Delta G_j(T)\) と \(k_{j,f}(T)\) が与えられたとき、熱力学整合性を満たす \(k_{j,r}(T)\) の計算手順を書け。
