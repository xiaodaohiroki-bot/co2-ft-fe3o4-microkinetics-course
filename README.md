# CO2-FT Fe3O4 Microkinetics Course

This repository is a course workspace for studying CO2 Fischer-Tropsch (CO2-FT) chemistry on Fe3O4-related catalysts with a focus on microkinetic modeling.

## Scope

- Build a practical learning path for CO2 activation, RWGS, carburization, and C-C coupling on iron oxide / iron carbide catalyst systems.
- Connect surface-science concepts with executable microkinetic models.
- Keep assumptions, parameters, and literature sources traceable enough for later research use.

## Planned Contents

- Course notes for CO2-FT reaction networks.
- Microkinetic modeling examples.
- Parameter tables and literature-derived assumptions.
- Exercises for sensitivity analysis, rate control, and catalyst-design interpretation.
