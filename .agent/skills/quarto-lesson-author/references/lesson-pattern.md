# Lesson Pattern

Use this pattern for new lessons in the CO2-FT / Fe3O4 microkinetics course.

```markdown
---
title: "Lesson title"
---

Opening paragraph that states the lesson purpose and the MVP boundary.

## 学習目標

- Goal 1
- Goal 2
- Goal 3

## Main technical section

Explain concepts with equations, short tables, and explicit notation.

## R1-R7との接続

Connect the concept to the minimal Fe3O4-RWGS mechanism without adding extra reactions.

## この章の要点

- Summary point 1
- Summary point 2
- Summary point 3

## 確認問題

1. Question 1
2. Question 2
3. Question 3
4. Question 4
5. Question 5

## 計算演習

1. Exercise 1
2. Exercise 2
3. Exercise 3
```

Do not put a body-level `# Lesson title` after the YAML header. Quarto renders the YAML title automatically.
